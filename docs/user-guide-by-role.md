# คู่มือการใช้งานระบบจัดครูสอนแทน แยกตามบทบาท

เอกสารนี้สรุปวิธีใช้งานระบบสำหรับผู้ใช้แต่ละบทบาท โดยอ้างอิงเมนูและสิทธิ์ในระบบปัจจุบัน สิทธิ์ของบางบทบาทสามารถปรับได้ที่เมนู `ตั้งค่าระบบ` > `ปรับสิทธิ์`

## เมนูหลักของระบบ

- `ภาพรวม`: ดูสรุปจำนวนครู คาบรอจัดสอนแทน แลกคาบรออนุมัติ และรายการสำคัญประจำวัน
- `บันทึกการลา/ไปราชการ`: บันทึกลาป่วย ลากิจ หรือไปราชการ และดูคาบที่เกี่ยวข้อง
- `จัดสอนแทน`: เลือกครูเข้าแทน ดูครูว่างในคาบนั้น และ export รูปสรุปสอนแทนตามสิทธิ์
- `แลกคาบ`: ดำเนินการเข้าแทนหรือสลับคาบตามเงื่อนไขของระบบ
- `สถิติ`: ดูรายงานและ export ข้อมูลรายงาน
- `อัพโหลดข้อมูล`: นำเข้า/แก้ไขข้อมูลครูและตารางสอน
- `ตั้งค่าระบบ`: จัดการผู้ใช้ สิทธิ์ ภาคเรียน สำรองข้อมูล กู้คืนข้อมูล และตั้งค่ารายวิชา

## ผู้ดูแลระบบ

บทบาท: `Admin`

ผู้ดูแลระบบมีสิทธิ์ครบทุกส่วนของระบบ เหมาะสำหรับผู้รับผิดชอบระบบหลักหรือผู้ดูแลข้อมูลโรงเรียน

### งานเริ่มต้นระบบ

1. เข้าสู่ระบบด้วยบัญชีผู้ดูแลระบบ
2. เข้าเมนู `อัพโหลดข้อมูล`
3. ตรวจหรือนำเข้าข้อมูลครูในหน้า `ข้อมูลครู`
4. ตรวจหรือนำเข้าตารางสอนในหน้า `ตารางสอน`
5. เข้าเมนู `ตั้งค่าระบบ` > `ตั้งค่ารายวิชา`
6. กำหนดรายวิชาที่ไม่ต้องจัดสอนแทน เช่น กิจกรรมหรือรายวิชาพิเศษ
7. ตั้งค่า `คาบประชุมระดับ` หากต้องการให้คาบประชุมระดับปรากฏในตารางครู

### จัดการผู้ใช้

1. เข้าเมนู `ตั้งค่าระบบ` > `จัดการผู้ใช้`
2. อนุมัติบัญชีที่ลงทะเบียนใหม่
3. ผูกบัญชีกับครูที่เกี่ยวข้อง
4. กำหนดบทบาท เช่น ครู หัวหน้างานบุคคล หัวหน้ากลุ่มสาระ หรือตัวแทนกลุ่มสาระ
5. หากต้องแก้ไขบัญชี กด `แก้ไข`
6. หากต้องการลบบัญชี กด `ลบผู้ใช้` และยืนยันในหน้าต่างที่เด้งขึ้นมา

หมายเหตุ: ระบบไม่อนุญาตให้ลบบัญชีที่กำลังใช้งานอยู่

### ปรับสิทธิ์

1. เข้าเมนู `ตั้งค่าระบบ` > `ปรับสิทธิ์`
2. เลือกเปิด/ปิดสิทธิ์ของแต่ละบทบาท
3. กดบันทึกการตั้งค่า

สิทธิ์ที่ปรับได้ เช่น:

- บันทึกลา/ไปราชการของตนเอง
- บันทึกลา/ไปราชการให้ทุกคน
- จัดครูสอนแทน
- จัดการแลกคาบ
- อนุมัติการเปลี่ยนแปลงคาบ
- ดู Dashboard สถิติ
- export รูปสรุปสอนแทนรายครู
- export รูปสรุปสอนแทนรายกลุ่มสาระ
- export รูปสรุปสอนแทนรายวัน
- จัดการข้อมูลครู
- จัดการตารางสอน

### สำรองข้อมูลและกู้คืนข้อมูล

สำรองข้อมูล:

1. เข้าเมนู `ตั้งค่าระบบ` > `สำรองข้อมูล`
2. กดดาวน์โหลดไฟล์ backup
3. เก็บไฟล์ไว้ในที่ปลอดภัย

กู้คืนข้อมูล:

1. เข้าเมนู `ตั้งค่าระบบ` > `กู้คืนข้อมูล`
2. เลือกไฟล์ backup `.json`
3. เลือกโหมดเติมข้อมูลที่ขาดหรือแทนที่ข้อมูลทั้งหมด
4. ยืนยันตามขั้นตอนบนหน้าจอ

### เริ่มภาคเรียนใหม่

1. เข้าเมนู `ตั้งค่าระบบ` > `เริ่มภาคเรียนใหม่`
2. กด `เริ่มภาคเรียนใหม่`
3. ระบบจะแจ้งเตือนให้สำรองข้อมูลก่อน
4. เลือก `สำรองข้อมูลทันที` หรือ `ฉันสำรองข้อมูลแล้ว`
5. เลือกภาคเรียนใหม่และโหมดตารางสอน
6. ติ๊กยืนยันว่าได้สำรองข้อมูลแล้ว
7. กด `ยืนยันเริ่มภาคเรียนใหม่`

โหมดที่ใช้ได้:

- `copy_schedule`: คัดลอกตารางสอนจากภาคเรียนเดิม
- `blank_schedule`: เริ่มจากตารางว่าง

## หัวหน้างานบุคคล

บทบาท: `หัวหน้างานบุคคล`

เหมาะสำหรับผู้ดูแลการลา/ไปราชการและการจัดสอนแทนในภาพรวม สิทธิ์เริ่มต้นคือบันทึกลาให้ทุกคน จัดสอนแทน และดูรายงาน

### บันทึกครูลาป่วย/ลากิจ/ไปราชการ

1. เข้าเมนู `บันทึกการลา/ไปราชการ`
2. เลือกวันที่
3. เลือกครู
4. กด `แสดงคาบสอน`
5. เลือกคาบที่เกี่ยวข้อง
6. เลือกประเภท เช่น ลาป่วย ลากิจ หรือไปราชการ
7. บันทึกข้อมูล

กรณีบันทึกหลายคน:

1. ไปที่ส่วน `บันทึกลาป่วย/ลากิจหลายคน`
2. เลือกวันที่
3. เลือกประเภท
4. ติ๊กครูหลายคน
5. กด `บันทึกหลายคน`

### จัดครูสอนแทน

1. เข้าเมนู `จัดสอนแทน`
2. เลือกวันที่หรือกลุ่มสาระตามที่ต้องการ
3. เลือกรายการคาบที่ต้องจัดแทน
4. ดูรายชื่อครูที่ว่างในคาบนั้น
5. ตรวจข้อมูลประกอบ เช่น จำนวนคาบที่สอนวันนี้ และประวัติเคยเข้าแทน
6. เลือกครูสอนแทน
7. บันทึกการจัดสอนแทน

หมายเหตุ:

- รายวิชาที่ตั้งค่าเป็น `ไม่ต้องจัดสอนแทน` จะไม่ถูกดึงมาจัดแทน
- คาบประชุมระดับไม่จำเป็นต้องจัดสอนแทนเมื่อครูลา

### ดูรายงาน

1. เข้าเมนู `สถิติ`
2. เลือกช่วงรายงาน เช่น รายวัน รายเดือน หรือรายภาคเรียน
3. เลือกกลุ่มสาระ หากต้องการแยกตามกลุ่มสาระ
4. ดูรายละเอียดการสอนแทนและภาระการเข้าแทน
5. export รายงานตามสิทธิ์ที่ได้รับ

## หัวหน้ากลุ่มสาระ

บทบาท: `หัวหน้ากลุ่มสาระ`

เหมาะสำหรับผู้รับผิดชอบการจัดสอนแทนและแลกคาบภายในกลุ่มสาระ สิทธิ์เริ่มต้นครอบคลุมจัดสอนแทน แลกคาบ อนุมัติการเปลี่ยนแปลงคาบ ดูรายงาน และ export รูปสรุปบางประเภท

### จัดสอนแทนในกลุ่มสาระ

1. เข้าเมนู `จัดสอนแทน`
2. ระบบจะแสดงข้อมูลตามขอบเขตกลุ่มสาระของผู้ใช้
3. เลือกคาบที่ต้องจัดแทน
4. เลือกครูที่ว่างในคาบนั้น
5. ตรวจจำนวนคาบสอนวันนี้และข้อมูลประกอบ
6. บันทึกครูสอนแทน

### Export รูปสรุปสอนแทน

1. เข้าเมนู `จัดสอนแทน`
2. ไปที่ส่วน `ปริ๊นต์เอาภาพสอนแทน`
3. เลือกช่วงวันที่
4. เลือกสรุปตามครูหรือกลุ่มสาระ
5. ตรวจว่าจัดคาบครบตามเงื่อนไข
6. กดแชร์เป็นรูปภาพหรือดาวน์โหลดรูปภาพ

เงื่อนไขสำคัญ:

- สรุปรายครู export ได้เมื่อจัดสอนแทนครูคนนั้นครบแล้ว
- สรุปรายกลุ่มสาระ export ได้เมื่อจัดสอนแทนครูที่ลาป่วยในกลุ่มสาระนั้นครบแล้ว
- สรุปรายวันขึ้นกับสิทธิ์ที่ผู้ดูแลระบบเปิดให้

### แลกคาบและอนุมัติการเปลี่ยนแปลงคาบ

1. เข้าเมนู `แลกคาบ`
2. เลือกรายการที่ต้องดำเนินการ
3. ตรวจว่าเป็นประเภท `เข้าแทน` หรือ `สลับคาบ`
4. ตรวจเงื่อนไขไม่ให้ครู ห้องเรียน หรือห้อง/อาคารชนกัน
5. พิจารณาอนุมัติหรือไม่อนุมัติรายการที่เกี่ยวข้อง

ข้อควรจำ:

- `เข้าแทน` คือครูอีกคนเข้าสอนแทนครูเจ้าของคาบในวันเดิม คาบเดิม ห้องเดิม
- `สลับคาบ` คือสลับตำแหน่งของคาบเรียน 2 คาบ และต้องอยู่ในห้องเรียนหรือ class group เดียวกัน
- คาบประชุมระดับและคาบกิจกรรมที่ระบบล็อกไว้ไม่ควรถูกนำไปแลกหรือสลับคาบ

## ตัวแทนกลุ่มสาระ

บทบาท: `ตัวแทนกลุ่มสาระ`

เหมาะสำหรับผู้ช่วยหัวหน้ากลุ่มสาระในการจัดสอนแทนและแลกคาบ สิทธิ์เริ่มต้นคล้ายหัวหน้ากลุ่มสาระ แต่โดยปกติอาจไม่มีสิทธิ์ดูรายงานรวม ขึ้นกับการตั้งค่าของผู้ดูแลระบบ

### งานหลัก

1. เข้าเมนู `จัดสอนแทน`
2. เลือกคาบที่ต้องจัดแทนในขอบเขตที่ได้รับสิทธิ์
3. เลือกครูที่ว่างและเหมาะสม
4. บันทึกครูสอนแทน
5. เข้าเมนู `แลกคาบ` เพื่อดำเนินการแลกคาบหรืออนุมัติรายการตามสิทธิ์

### Export รูปสรุป

1. เข้าเมนู `จัดสอนแทน`
2. ไปที่ `ปริ๊นต์เอาภาพสอนแทน`
3. เลือกครูหรือกลุ่มสาระ
4. กด export เมื่อระบบจัดครบตามเงื่อนไข

หากไม่เห็นปุ่ม export ให้แจ้งผู้ดูแลระบบตรวจสิทธิ์ในหน้า `ปรับสิทธิ์`

## ครู

บทบาท: `ครู`

เหมาะสำหรับผู้ใช้ทั่วไปที่ต้องบันทึกข้อมูลของตนเองและดำเนินการแลกคาบตามสิทธิ์ สิทธิ์เริ่มต้นคือบันทึกลา/ไปราชการของตนเองและจัดการแลกคาบ

### ลงทะเบียนและเข้าสู่ระบบ

1. เปิดหน้าเข้าสู่ระบบ
2. หากยังไม่มีบัญชี ให้กด `ลงทะเบียน`
3. กำหนด username และ password
4. เลือกเชื่อมโยงกับครูของตนเอง หากระบบเปิดให้เลือก
5. รอผู้ดูแลระบบอนุมัติบัญชี
6. เมื่ออนุมัติแล้วจึงเข้าสู่ระบบได้

### บันทึกลากิจหรือไปราชการของตนเอง

1. เข้าเมนู `บันทึกการลา/ไปราชการ`
2. เลือกวันที่
3. ระบบจะแสดงคาบสอนของตนเอง
4. เลือกคาบที่เกี่ยวข้อง
5. เลือกประเภท เช่น ลากิจหรือไปราชการ
6. บันทึกรายการ

หมายเหตุ: การลาป่วยอาจถูกบันทึกโดยหัวหน้างานบุคคลหรือผู้มีสิทธิ์จัดการข้อมูลการลา

### แลกคาบ

1. เข้าเมนู `แลกคาบ`
2. เลือกรายการคาบที่เกี่ยวข้อง
3. เลือกประเภทการเปลี่ยนแปลง
4. ตรวจ preview ก่อนยืนยัน
5. ส่งคำขอ
6. รอการอนุมัติจากผู้มีสิทธิ์

### ตรวจรายการที่เกี่ยวข้องกับตนเอง

1. ดูหน้า `ภาพรวม` เพื่อดูรายการที่ต้องดำเนินการ
2. เข้าเมนู `แลกคาบ` เพื่อตรวจรายการที่รออนุมัติหรือเกี่ยวข้องกับตนเอง
3. หากเป็นครูปลายทางของคำขอ ให้พิจารณารายการตามสิทธิ์ที่ระบบกำหนด

## การทำงานประจำวันแบบย่อ

### กรณีมีครูลาป่วย

1. หัวหน้างานบุคคลบันทึกลาป่วยที่หน้า `บันทึกการลา/ไปราชการ`
2. ระบบสร้างรายการคาบที่ต้องจัดแทนตามตารางสอน
3. หัวหน้างานบุคคล หัวหน้ากลุ่มสาระ หรือตัวแทนกลุ่มสาระเข้าเมนู `จัดสอนแทน`
4. เลือกครูว่างในคาบนั้น
5. บันทึกครูสอนแทน
6. export รูปสรุปหากต้องการประกาศหรือปริ๊นต์

### กรณีครูลากิจหรือไปราชการ

1. ครูหรือผู้มีสิทธิ์บันทึกรายการที่หน้า `บันทึกการลา/ไปราชการ`
2. ตรวจคาบที่เกี่ยวข้อง
3. ดำเนินการแลกคาบหรือจัดการตามแนวปฏิบัติของโรงเรียน
4. ผู้มีสิทธิ์อนุมัติรายการในหน้า `แลกคาบ`

### กรณีต้องเริ่มภาคเรียนใหม่

1. ผู้ดูแลระบบสำรองข้อมูลก่อนเสมอ
2. เข้าเมนู `ตั้งค่าระบบ` > `เริ่มภาคเรียนใหม่`
3. เลือกภาคเรียนใหม่
4. เลือกว่าจะคัดลอกตารางสอนหรือเริ่มตารางว่าง
5. ยืนยันว่าได้สำรองข้อมูลแล้ว
6. กดเริ่มภาคเรียนใหม่

## ข้อควรระวัง

- ห้ามรันคำสั่ง seed กับฐานข้อมูลจริง เพราะอาจล้างข้อมูล demo ทับข้อมูลใช้งาน
- ก่อนเริ่มภาคเรียนใหม่หรือ reset ระบบ ควรสำรองข้อมูลก่อนทุกครั้ง
- ไม่ควรแชร์บัญชีผู้ดูแลระบบให้ผู้ใช้งานทั่วไป
- หากเมนูไม่แสดง อาจเกิดจากยังไม่ได้รับสิทธิ์หรือบัญชียังไม่ถูกอนุมัติ
- หากปุ่ม export ไม่แสดง ให้ตรวจสิทธิ์ที่เมนู `ตั้งค่าระบบ` > `ปรับสิทธิ์`
- หาก UI ในเครื่องหายหลัง build/dev หลายรอบ ให้หยุด dev server ที่ค้าง ล้าง `.next` แล้วเปิด `npm run dev` ใหม่
